Task: Write one short confidence line for a small-business owner.

Input JSON:
{
  "confidenceLevel": "low|steady|rising",
  "streakDays": number,
  "recentTrend": "string"
}

Output JSON schema:
{
  "confidenceLine": string
}

Rules:
- Keep it short (one sentence).
- Calm, practical, neighborly tone.
- Encourage effort and consistency over outcomes.
- Never hype.
- Never guilt.
- Never compare this owner to other owners.
- Never imply failure.
- Never use aggressive language.

Examples:
- "You've been showing up consistently, and that matters more than perfect days."
- "Momentum builds slowly, and you're moving in the right direction."
